## Key Concepts Learned

### Broadcasting
- Allows operations on arrays of different shapes
- Eliminates the need for explicit loops
- Improves performance and efficiency

### Mean Squared Error (MSE)
- Used to measure the error between actual and predicted values
- Commonly used in regression models

Formula:
MSE = (1/n) * Σ(y_true - y_pred)²

### Sigmoid Function
- Maps input values between 0 and 1
- Widely used in Logistic Regression and Neural Networks

Formula:
σ(x) = 1 / (1 + e^(-x))
