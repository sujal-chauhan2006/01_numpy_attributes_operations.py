## Key Concepts Learned

### Broadcasting

* Allows operations on arrays of different shapes
* Eliminates the need for explicit loops
* Improves performance

---

### Mean Squared Error (MSE)

* Measures difference between actual and predicted values
* Used in regression problems

Formula:
MSE = (1/n) × Σ(y_true − y_pred)²

---

### Sigmoid Function

* Maps values between 0 and 1
* Used in Logistic Regression and Neural Networks

Formula:
σ(x) = 1 / (1 + e^(-x))
